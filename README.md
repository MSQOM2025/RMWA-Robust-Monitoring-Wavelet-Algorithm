# RMWA-Robust-Monitoring-Wavelet-Algorithm
Official implementation of the Robust Monitoring Wavelet Algorithm (RMWA) for solving Stochastic Itô-Volterra Integral Equations under heavy-tailed Lévy-type noise. Includes adaptive stability monitoring ( S(M)  index), Huber-weighted IR-LSQR solvers, and benchmarking against Tikhonov and TSVD regularizations.
